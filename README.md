# Pasticceria Certosa — Artisan Bakery Website

> Client website for an artisan bakery in Milan. Showcases products, story, and contact — designed for local SEO and mobile customers.

---

## Overview

A modern, mobile-first website for a traditional Italian bakery. Clean design that highlights the artisanal identity of the brand, with product showcases, opening hours, and a contact section optimised for local search.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 16 (App Router) + React 19 + TypeScript |
| Styling | Tailwind CSS 3.4 |
| Backend | Supabase (auth + data) |
| Deployment | Vercel |

---

## Local Setup

```bash
git clone https://github.com/marcoantoniovillalva-bot/pasticceria-certosa
cd pasticceria-certosa
npm install
cp .env.local.example .env.local
npm run dev
```
